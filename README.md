# 电影日志
## Github页面部署：nathanlilxx.github.io
## 一、网站说明
### 1. 登陆权限
- 查看“电影集”及“影评集”不需要登录标识
- “创建”功能需要登录标识
  - <b>cookie</b>设置登录标识符（有效期一小时）
### 2. 功能开发
- “电影集”网页展示电影列表
  - 列表采用<b>grid</b>进行网格布局
  - 电影列表中的数据由“创建”功能生成
    - 数据采用本地存储（<b>local storage</b>）
    - 页面元素的动态增删改由<b>DOM API</b>实现

  - “筛选”按钮根据电影类型动态展示电影列表
    
- 影评页面展示字段排列
  - 影评部分为静态页面，根据设备宽度<b>响应式功能</b>
